# CampaMapa

App para agregar amigos/campistas y ubicarlos en un mapa de Colombia. Cada persona se registra con su municipio y un nivel, y aparece como marcador en el mapa.

## Stack

- React + Vite
- react-router-dom
- react-leaflet / Leaflet
- Tailwind CSS
- Persistencia en `localStorage` (sin backend por ahora)

## Desarrollo

```bash
pnpm install
pnpm dev       # servidor de desarrollo
pnpm build     # build de producción
pnpm lint      # eslint
pnpm preview   # sirve el build de producción localmente
```
